**START LOCALLY AND PUSH TO GITHUB**
1. git init => creates .git directory
2. git add . => stages all files to commit
3. git commit -m "Initial Commit" =>  saves snap shot "-m" => message
4. git branch -M main => "-M" force rename" old git uses master => optional
5. git remote add origin <repo url> => origin => nickname for remote repo, => connects local repo to remote repo
6. git push -u origin main => sends commits to remote => push to origin(remote repo) main branch => -u set upstream
   
**START ON GITHUB AND CLONE LOCALLY**
1. git clone <repo-url>
2. cd repo-name
3. # start coding 
4. git add .
5. git commit -m "Initial Commit"
6. git push
